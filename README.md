# Casibom SEO Sitesi

Casibom hakkında SEO odaklı, modern ve responsive web sitesi.

## Özellikler

- ✅ SEO Optimizasyonu (Meta tags, Structured Data, Sitemap)
- ✅ Turuncu-Siyah Modern Tema
- ✅ Responsive Tasarım (Mobil Uyumlu)
- ✅ Favicon Paketi
- ✅ Modern Animasyonlar ve Efektler
- ✅ FAQ Bölümü
- ✅ İstatistikler ve Özellikler

## Dosyalar

- `index.html` - Ana sayfa
- `giris.html` - Casibom giriş sayfası
- `telegram.html` - Casibom telegram sayfası
- `twitter.html` - Casibom twitter sayfası
- `hakkimizda.html` - Hakkımızda sayfası
- `styles.css` - CSS stilleri
- `script.js` - JavaScript dosyası
- `favicon.svg` - SVG favicon
- `site.webmanifest` - Web manifest dosyası
- `robots.txt` - SEO için robots dosyası
- `sitemap.xml` - Site haritası

## Favicon Dosyaları

Aşağıdaki favicon dosyalarını oluşturmanız gerekmektedir (turuncu-siyah tema):

- `favicon.ico` - 16x16, 32x32, 48x48
- `favicon-96x96.png` - 96x96 PNG
- `apple-touch-icon.png` - 180x180 PNG
- `favicon.svg` - SVG format (zaten oluşturuldu)

## SEO Anahtar Kelimeler

- casibom
- casibom giriş
- casibom telegram
- casibom twitter
- casibom giriş adresi
- casibom bahis

## Kullanım

1. Tüm dosyaları web sunucunuza yükleyin
2. Favicon dosyalarını oluşturup yükleyin
3. `site.webmanifest` dosyasındaki URL'leri güncelleyin
4. `sitemap.xml` dosyasındaki URL'leri güncelleyin
